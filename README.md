##使用TS开发Vue项目时，关于使用OptionsAPI定义Vue组件描述正确的是 （ABCD）  
A. 组件仍然可以使用以前的方式定义(到处组件选项对象，或者使用Vue.extend())┋
B. 导出的是一个普通对象时，TS无法推断出对应的类型┋
C. VSCode可以推断出类型成员是因为使用了Vue插件┋
D. 必须使用 vue.extend() 方法确保TS能够有正常的类型判断
